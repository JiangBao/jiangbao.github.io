代码那些事

  • 可视化一键生成 Nginx 配置，打开网站，按照需求填写参数，系统自动生成配置文件，体验一下吧
  • 30 天 JavaScript 编程挑战，30 天学习 JavaScript 的分步指南，结合编码实践的学习总是要更有趣且更高效一些
  • Dan 在 reddit 上关于 React18 中请求数据的推荐方式的回答，简单说，这是存在大部分 useEffect/componentDidMount/onMount 前端组件框架的共性问题，目前的「推荐方式」，需要在清除函数中设置状态标记解决竟态问题
  • Bun 发布 beta 版本，Bun 是一个 all-in-one 的 JavaScript 运行环境，类似之前的 Node.js 和 Deno，使用 Zig 编写，主要专注于：
    • 更快的启动速度
    • 更强的运行性能
    • 更完善的工具库：打包、编译、包管理等等
