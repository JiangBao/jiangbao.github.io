代码那些事

  • 谷歌发布新的开源操作系统 KataOS，用于进行机器学习的嵌入式设备的开源操作系统，几乎全由 Rust 实现，新坑来了。
  • 一份 JavaScript 使用报告，从加载数量、请求数，到常用库等等分析，前端从业者可以看看。最近在做一个历史项目的性能优化工作，对于文中提到的「大量被加载但是未使用的 JavaScript」深有感触。
  • React 渲染的未来，讨论了 React 当前的渲染模式、它们存在的问题，以及 React 18 引入的新模式如何是解决这些问题的。
  • 微前端如何做样式隔离，目前比较常用的还是 css Module + 协商命名规则，本文也讨论了 Shadow DOM 实现完全隔离样式。
  • ViteConf 2022 回顾，可以看看尤雨溪《How Vite Came to Be》的演讲，了解 Vite 的诞生历程。
