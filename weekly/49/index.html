兴趣周刊(第 49 期)

代码那些事

  • React，visualized
    一个非常棒的 React 可视化教程，目前包含了 React 背景、状态管理，副作用等核心概念。交互式的可视化配图很不错，如果觉得官方文档太过单调，可以看看这个。👍🏻

  • 基于 fabric.js 和 Vue 的图片编辑器
    面向非专业设计人员的图片编辑器，可以用来快速做海报、头图之类的。之前使用 Fabric.js 开发一个项目的 canvas 交互功能时，看过这个项目的代码，恰好在 V 站看到了此 V 友 发帖，支持一个。

  • Node.js 发布 v20 版本
    2023.04.18, Node.js 20.0.0 版本发布了，具体更新可以看看 changelog。有一项实验特性：打包单个可执行文件，打包成二进制文件，在未安装 Node.js 的机器上也可以运行，不过目前貌似最基础的包都很大，一个简单的 hello world 打包完有 85M。

  • 在 web 上使用字体
    逛 V 站的时候刷到这样一个 帖子，讨论大家目前使用的字体，其中看到了这篇关于 web 字体的文章，整理得很好，也了解到了是乔布斯和苹果最早将字体带进了数码世界。
    另外，看到大家都在推荐 霞鹜文楷，我也将自己的站点换了 LXGW WenKai Mono 字体，看着挺不错的。

其它的关注

  • 比亚迪仰望 U8 开启预售，预售价 109.8 万元
    真正的国产高端车型，「仰望」这名字就取的挺好的 🤔。
    之前发布会时，大秀肌肉，易四方、云辇-P、原地掉头、应急浮水等等，看得眼花缭乱，坐等第一批车主的测评视频。

  • 苹果 Apple Card 储蓄账户上线，年利率 4.15%
    当地时间 4 月 17 日起，美国本土苹果用户可以在 iPhone 钱包中开设高盛的高收益储蓄账户，并获取利息，这是苹果利用自己广大的用户群体，新开了一个「余额宝」项目吗？

  • 《灌篮高手》电影国内上映，人气火爆
    主要内容是湘北高中与山王工业的全国大赛对决，漫迷早已知道结局，但大部分还是选择去为热血的青春补张票。
    年纪大了，画风变了，感觉当年动画版，停留在大家前往全国大赛的列车上，挺好的。
