代码那些事

  • 属于 Stack Overflow 的时代过去了
    Stack Overflow 提问数量已经低于 2008 年上线首月的数量，曾经作为程序员的重要技能之一：善用 Stack Overflow，已经随着这波 AI 浪潮，被重塑了

  • Tailwind 总裁透露公司因 AI 冲击陷入严重财务危机，已裁员 75%
    本周 Tailwind CSS 创始人兼 CEO Adam Wathan 拒绝了一个为文档添加 llms.txt 接口的 GitHub PR，并回复称公司因 AI 冲击陷入严重财务危机，已裁员 75%，项目仅余 6 个月资金储备
    与上条消息一样，同样是 AI 浪潮冲击，但 Tailwind 作为许多大模型生成代码默认会使用的开源组件库，更让人思考，未来 AI 时代，开源协议是不是已经荡然无存了
    好消息是在这条新闻发酵后，Tailwind CSS 迅速获得了包括谷歌 AI Studio、Cursor 等多家科技公司以及许多个人开发者的新赞助，AI 时代开源项目甚至技术工作者如何找到可持续的商业模式，避免这样的危机，这个阶段似乎每个人都在焦虑地探索

  • How Terminals work
    一个有趣的站点，帮助理解终端工作原理，进而解释了最近火热的 TUI 原理，并且整个站点是由一个产品设计师通过 Claude 生成

其它的关注

  • 本周观影：浪浪人生
    之前没有去电影院看，本周在网络平台看完了，普通的家庭温情片，阿太说的那句：「只要好好活着，就已经赢了，干嘛非得出人头地」是想升华的主题，但是整体缺少了一些逻辑，有些刻意营造苦难了。与此类似的主题，更喜欢《心灵奇旅》的：当你想认真生活的那一刻，生命的火花就已经点亮了
    不过其中有个片段：儿子黄远达在医院照顾父亲黄荣发，结果隔壁床的病友抢救无效走了，相同的一幕几年前在医院照顾病重的父亲时也遇到了，对那种绝望的感觉感同身受
    听说原著《皮囊》不错，找时间读读
